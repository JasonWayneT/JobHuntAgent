# Design Spec: DESIGN-001 Applyr UI

## Metadata

- Design spec ID: `DESIGN-001`
- Status: implemented
- Related feature specs: `FEAT-006`
- Related requirements: `DES-001` through `DES-005`

## Context

The UI follows the "Applyr" design system: soft minimalism, nature-rooted colors, and editorial typography.

## Style Tokens

- **Primary:** Sage Green (#526452)
- **Secondary:** Muted Terracotta (#8b4f3a)
- **Surface:** Warm Stone Neutral (#faf9f6)
- **Typography:** Manrope (Headings), Inter (Body)

## Layout Philosophy

- **No-Line Rule:** Borders are prohibited; depth is created via background color shifts.
- **Intentional Asymmetry:** Avoid rigid grids for a more organic, premium feel.

## Component Design

- **Primary Button:** Gradient from Sage Green to Primary Dim.
- **Application Card:** Borderless, 12px corner radius, 24px vertical whitespace for separation.
